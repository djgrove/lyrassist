About

LyrAssist was created by Devon Grove.Artist bios and images are graciously provided by Last.fm's API, and lyrics data courtesy of Genius.

As of now, all artist lyrics are generated using a Markov model, though in the future I will be looking to introduce alternate models for more accurate lyrics. I will be adding functionality and new artists on an ongoing basis, so check back for new content soon!

  • This site is serverless, running using AWS Lambda and S3!
  • Please note this project is not-for-profit and is not intended for commercial use.
